## Исходные данные (Excel)

- **Файл:** `.xlsx` с результатами тестов, английский язык, 9 класс
- **Строка 4:** максимальные баллы за каждое задание
- **Строка 5+:** данные учеников
- **Колонки 0–1:** номер и имя ученика
- **Колонки 2–12:** задания 1–11 (Аудирование)
- **Колонки 13–20:** задания 12–19 (Чтение)
- **Колонки 21–28:** задания 20–28 (Грамматика)
- **Колонки 29–35:** задания 29–34 (Словообразование)
- **Колонки 37–41:** Письмо (2-я часть, 5 заданий)
- **Колонки 42–51:** итоги: sum1, pct1, sum2, pct2, итого балл, итого%, устная сумма, устная%, полный балл, полный%
- **Максимумы:** Часть 1 = 43 б, Часть 2 = 10 б, Итого письменная = 53 б

### Разделы и колонки (0-based)
| Раздел | Задания | slice | Макс |
|--------|---------|-------|------|
| 🎧 Аудирование | 1–11 | `[2:13]` | 15 б |
| 📖 Чтение | 12–19 | `[13:21]` | 13 б |
| ✏️ Грамматика | 20–28 | `[21:29]` | 9 б |
| 🔤 Словообразование | 29–34 | `[29:36]` | 6 б |
| ✍️ Письмо | — | `[37:42]` | 15 б |

Секционный % в данных: `tasks[slice]`, максимум из строки 4.  
`sec_pct = sum(баллов) / max * 100`  
`avg_sec` = среднее арифметическое % по всем тестам с данными.

### Листы и порядок
- Лист `шаблон` — **пропускается**
- Листы в файле идут в **обратном** хронологическом порядке → `reversed(wb.sheetnames)`
- Лист `ЭКЗАМЕН` / `ЭКЗАМЕН (отработка)` — **школьный пробник**, не настоящий ОГЭ
  - Отображается как **обычный тест** в хронологическом порядке
  - Никакого специального выделения, никакой особой точки на графике
  - Подпись берётся из имени листа

### Группы
- Сброс нумерации (`n < prev_num` и `n <= 2`) = новая группа
- Строка «Среднее значение» = конец группы
- Нормализация имён: `name.strip().lower().split('(')[0].strip()`

### Блок рекомендаций — правила (ВАЖНО)
- **✅ Сильные:** разделы ≥85%; если нет — лучший раздел
- **⚠ Зоны роста:** ВСЕГДА непустые:
  - `Критично` — раздел <70%
  - `Нестабильно` — раздел 70–85%
  - `Потенциал роста` — если всё ≥85%, показать самый слабый
  - `Разброс X%` — если разброс между тестами ≥20%
  - **«Всё в норме» — никогда не писать**
- **📈 Тренд:** `первый%→последний%`, добавить `⚠−X%` при просадке ≥10%
- **🎯 Рекомендация:** конкретные действия по 1–2 слабым разделам

## Вариант 2 — PDF для Ч/Б печати (A4, один ученик = один лист)

### Технология
1. Python генерирует **print-HTML** (только десктоп, вёрстка на `<table>`)
2. Замеряем высоту каждой страницы через **Playwright**:
```javascript
const heights = await page.evaluate(() =>
  Array.from(document.querySelectorAll('.page')).map(p => p.scrollHeight)
);
// zoom = 1062 / Math.max(...heights)  (1062px = A4 рабочая высота при 96dpi, margin 8mm)
```
3. Конвертируем через **wkhtmltopdf**:
```bash
wkhtmltopdf \
  --page-size A4 --margin-top 8mm --margin-bottom 8mm \
  --margin-left 10mm --margin-right 10mm \
  --encoding utf-8 --enable-local-file-access \
  --zoom ZOOM --dpi 96 \
  input.html output.pdf
```
При 23 учениках высота контента ~630–730px → zoom ≈ 1.46

### Вёрстка print-HTML (table-based — wkhtmltopdf не любит flexbox)
```
<div class="page">
  1. <table> шапка: имя+группа слева | балл+средний справа (фон #111, белый текст)
  2. <table> 5 карточек (kcard): последний, %, средний, лучший, тренд
  3. <table> графики: бары по месяцам (160px) | SVG (остаток)
  4. <table> данные: таблица тестов (50%) | разделы + выводы (50%)
</div>
```

### CSS Ч/Б градация
```css
.gg { border-top: 3.5px solid #000; }   /* ≥85% чёрный */
.gb { border-top: 3.5px solid #333; }   /* ≥70% тёмно-серый */
.gw { border-top: 3.5px dashed #666; }  /* ≥55% пунктир */
.gr { border-top: 3.5px dotted #999; }  /* <55%  точки */

.bi.gg { background: #111; }
.bi.gb { background: #444; }
.bi.gw { background: #777; }
.bi.gr { background: #bbb; }
```

### Блоки выводов (print)
```css
.rg { background:#ebebeb; border-left:3px solid #000; }    /* сильные */
.rr { background:#f0f0f0; border-left:3px solid #444; }    /* зоны роста */
.rb { background:#eeeeee; border-left:3px dashed #666; }   /* тренд */
.ry { background:#f5f5f5; border-left:3px dotted #888; }   /* рекомендация */
```

### SVG (print-версия)
- viewBox: `0 0 500 130` (компактнее)
- stroke: `#111`, fill: `#333` opacity 0.08, сетка `#ccc`, средняя `#777`
- Точки: `r=3.5`, подписи font-size 8px

### Бары по месяцам (print, через table)
Группировка по меткам из имён листов:
```python
if   '30.10' in lbl: m='Окт'
elif '29.11' in lbl: m='Ноя'
elif '14.01' in lbl or '21.01' in lbl: m='Янв'
elif any(x in lbl for x in ['7.02','2-6.02','9-13.02','16-20.02','23-28.02']): m='Фев'
elif '02-08.03' in lbl: m='Мар'
elif 'ЭКЗАМЕН' in lbl: m='ЭКЗ'
```
Высота бар-зоны: 58px. Строить через `<table>` с `height` на `<td>`.

### @page

```css
@media print {
  @page { size: A4 portrait; margin: 8mm 10mm; }
  .page { width:100%; padding:0; margin:0; page-break-after:always; }
  .page:last-child { page-break-after:avoid; }
}
```
